# Zakimoto-Git-Tutorial-Kai

* 利用しているツール，ライブラリ

 * バックエンド  
  * Java
  * Spring boot
  
 * フロントエンド
  * jQuery
  * jQuery-console
  
 * データベース
  * MongoDB  

### 実行方法

* MongoDBの起動

```
mongod
```

* 実行可能jarの作成

```
./mvnw clean package
```

* jarの実行コマンド

```
java -jar target/zakimoto-git-tutorial-kai-0.0.1-SNAPSHOT.jar 
```
